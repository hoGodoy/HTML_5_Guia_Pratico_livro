Esse fildset contém está setado pra carregar automaticamente em solterio

Esse formulário não permite alterar dados do campo Tel, está setado como disabled. Há uma label no botão enviar.

No exemplo, no arquivo recebe.jsp, que recebe os dados do formulário no servidor, o conteú- do dos campos nomeados como “nome” e “telefone” pode ser recuperado para, por exemplo, ser armazenado em um banco de dados.

Para recuperar esses dados, no arquivo recebe.jsp (que é um arquivo JavaServer Pages) deve ser utilizado o seguinte scriptlet Java:

            
                <%
                String n = request.getParameter("nome");
                String t = request.getParameter("telefone");
                %>

Esse formulário traz alguns campos como somente leitura (readonly). É possível selecionar a informação, diferente do atributo disabled
